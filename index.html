Como Jogar

Descubra o país secreto. Você pode tentar quantas vezes precisar.

Depois de enviar uma palavra, você poderá visualizar no display se alguma das letras coincidem com a palavra secreta.

Quando a palavra é descoberta aparece uma mensagem de vitória juntamente com o número de tentativas que usou para tentar descobrir a palavra.
